QUEUE_MEMBER_LIST()

Synopsis

Returns a list of interfaces on a queue.

Description

Returns a comma-separated list of members associated with the specified queuename.

Syntax

QUEUE_MEMBER_LIST(queuename)
Arguments
  • queuename
