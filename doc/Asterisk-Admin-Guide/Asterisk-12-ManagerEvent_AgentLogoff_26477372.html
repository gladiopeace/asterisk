AgentLogoff

Synopsis

Raised when an Agent has logged off.

Description

Syntax

Action: 
Agent: <value>
Logintime: <value>
Arguments
  • Agent - Agent ID of the agent.
  • Logintime - The number of seconds the agent was logged in.
