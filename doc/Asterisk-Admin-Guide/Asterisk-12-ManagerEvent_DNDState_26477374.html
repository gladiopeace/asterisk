DNDState

Synopsis

Raised when the Do Not Disturb state is changed on a DAHDI channel.

Description

Syntax

Action: 
DAHDIChannel: <value>
Status: <value>
Arguments
  • DAHDIChannel - The DAHDI channel on which DND status changed.

    Note

    Icon

    This is not an Asterisk channel identifier.

  • Status
    • enabled
    • disabled
